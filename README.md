# vue-valid

> Vue表单校验解决方案

## 项目需要使用的命令

``` bash
# 安装依赖
npm install

# 启动调试服务器，即可边开发边预览
npm run dev

# 编译生产版本，开发完成后，可进行此步骤，将程序发布到服务器上
npm run build

# 运行单元测试
npm run unit

# 运行e2e测试
npm run e2e

# 运行所有测试
npm test
```

需要更详细地了解Vue编译工具，可参见 [webpack](https://github.com/vuejs-templates/webpack#vue-webpack-boilerplate) 与 [vue-loader](http://vuejs.github.io/vue-loader).
